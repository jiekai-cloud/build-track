這是一個全自動的系統檢測流程。執行此命令時，AI 助理會自動啟動伺服器並派駐測試代理程式進行深度瀏覽點擊，您 **全程無需手動同意**。

### 執行步驟：
1. 若系統目前沒有啟動，透過 `npm run dev` 啟動本地開發伺服器。
2. 呼叫 `browser_subagent` 工具，進入伺服器預設的本機網址 (例如：`http://localhost:5013/build-track/`)。
3. 代理程式會自動：
   - 點擊「體驗帳戶」。
   - 瀏覽左側欄位：儀表板、行事曆、專案、報價、人員等各大模組。
   - 深入檢查新建的頁籤或按鈕，尋找任何報錯或版面異常。
4. 代理程式測試完成後，直接總結一份「健康度與錯誤掃描報告」給使用者。
