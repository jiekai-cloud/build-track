這是一個全自動的系統檢測流程。使用 Playwright 測試框架，**全程無需手動同意**。

### 執行步驟：
1. 執行 `npx playwright test --reporter=list` 啟動 E2E 測試（Playwright 會自動啟動 dev server）。
2. 測試腳本會自動：
   - 以訪客模式登入系統
   - 逐一瀏覽所有側邊欄模組（儀表板、行事曆、專案、報價、派工、客戶、團隊、廠商、庫存、考勤、薪資、簽核、待辦等）
   - 檢查每個頁面是否正常載入、有無 JS 錯誤或版面異常
   - 進入專案詳情頁驗證 tab 名稱與順序
3. 測試完成後，根據終端機輸出的結果，整理一份「健康度與錯誤掃描報告」給使用者。
4. 如果有測試失敗，分析失敗原因並提出修復建議。
