# Salary System Design (薪資系統設計規範)

這份文件定義了本系統薪資計算的邏輯與規範，包含各類員工的薪資結構、津貼發放、假勤規則及特殊負擔計算。

## 1. 通用原則 (General Principles)

### 1.1 工時與日曆
*   **標準工時**：08:00 - 17:00 (每日 8 小時)。
*   **午休時間**：12:00 - 13:00 (1 小時，不計薪)。
*   **工作日**：原則上週休二日 (週一至週五)，依行政院人事行政總處行事曆。
*   **計薪基礎**：所有人員薪資計算皆需依據實際打卡記錄 (Clock-in/Clock-out)。

### 1.2 獎勵與津貼 (Bonuses & Allowances)
*   **午餐獎勵 (Lunch Bonus)**：
    *   **性質**：雖為「補助」，但在系統與薪資單上必須顯示為 **「獎勵金」 (Bonus)**，而非本薪或固定津貼。
    *   **目的**：便於獎懲制度執行（違規時可扣除此項目而不違反勞基法本薪規定）。
    *   **金額**：依職位階級設定不同金額。
*   **工業繩索技術津貼 (Rope Access Allowance)**：
    *   針對技術員施工日的額外補貼。
    *   **使用個人裝備**：+1500 元/日。
    *   **使用公司裝備**：+1200 元/日。

## 2. 員工類型與計算邏輯 (Employee Types)

### 2.1 月薪人員 (Monthly Salary)
*   **基本邏輯**：固定月薪，依打卡記錄檢核出勤。
*   **請假規則**：依照勞基法規定扣薪（病假半薪、事假不給薪等）。
*   **加班費**：依照勞基法標準計算（前2小時 1.33倍，後2小時 1.66倍...）。

### 2.2 日薪人員 (Daily Salary)
*   **基本邏輯**： **有做有錢，沒做沒錢 (No work, no pay)**。
*   **薪資費率**：每人每日薪資不同。
*   **特休規則**：目前設定為 **無特休**。
*   **假日給薪**：除非另有規定，否則國定假日不出勤不給薪。

### 2.3 時薪人員 (Hourly Salary)
*   類型分為「外包/承攬 (Contractor)」與「一般時薪 (Regular Hourly)」。
*   **特休規則 (Regular Hourly Only)**：
    *   有特休，但需**依照工時比例折算**。
    *   公式：`特休時數 = 全職特休時數 * (實際工作時數 / 全職標準工時)`。
    *   範例：若一個月工時僅為全職的一半，則該月累積特休為全職的一半。

## 3. 特殊排班與負擔計算 (Special Schedule & Insurance)

針對每週固定工作日少於 5 天的人員（如週休三日、四日者）：

### 3.1 特別事假 (Special Personal Leave)
*   **定義**：針對每週「固定未出勤」的工作日（例如固定週五不上班）。
*   **性質**：雖然勞基法事假上限 14 天，但此類固定休假不計入該上限，系統需設立專用的 **「特別事假」**。
*   **給薪**：**不支薪 (No Pay)**。

### 3.2 勞健保/勞退 額外負擔 (Additional Insurance Burden)
因工時不足導致公司仍需負擔全額保險費顯得不公平，故依比例轉嫁給員工：

| 每週工作天數 | 員工額外負擔 (佔公司應負擔金額 %) | 說明 |
| :--- | :--- | :--- |
| **4 天** | **15%** | 員工需支付：(自付額) + (公司負擔額 × 15%) |
| **< 4 天** | **40%** | 員工需支付：(自付額) + (公司負擔額 × 40%) |

*   **計算方式**：系統需在計算薪資時，自動計算公司負擔總額（勞保+健保+勞退），依比例計算出「代扣款」，並從薪資中扣除。

## 4. 系統實作需求 (System Requirements)

### 4.1 資料結構欄位
*   `TeamMember` 需新增欄位：
    *   `workDaysPerWeek`: 每週工作天數 (4, 5, or others)。
    *   `lunchBonus`: 午餐獎勵金額。
    *   `ropeLicenseLevel`: 繩索證照等級 (影響派工，可能影響津貼)。
    *   `insuranceBurdenRate`: 自定義負擔比率 (若未設定則依週工作日自動判定 15% 或 40%)。
*   `AttendanceRecord` (每日記錄) 需新增：
    *   `ropeEquipment`: 'personal' | 'company' | null (當日使用裝備類型)。
*   `PayrollRecord` (月薪資單) 需顯示：
    *   `lunchBonus`: 午餐獎勵小計。
    *   `ropeAllowance`: 繩索津貼小計。
    *   `insuranceExtraDeduction`: 特殊負擔扣款。

### 4.2 假別設定
在系統設定中新增假別：
*   **特別事假 (Special Personal Leave)**：
    *   扣薪比例：100% (不給薪)。
    *   不計入年度事假 14 天額度。
